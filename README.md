# Cryptography and Network Security Lab

Welcome to the Cryptography and Network Security Lab repository! This repository contains hands-on lab exercises and code for implementing various cryptographic algorithms and network security protocols.

## Contents

- **Lab Exercises:** Detailed lab instructions for cryptography and network security topics.
- **Code Samples:** Sample code for encryption, hashing, key exchange, and security protocols.
- **Packet Analysis:** Tools and methods for analyzing network traffic.

## Lab Syllabus

### Symmetric Encryption with DES
- **Objective:** Implement DES encryption and decryption.
- **Details:**
  - Use a 64-bit key size and 64-bit block size.
  - Verify encryption and decryption processes.

### Symmetric Encryption with AES
- **Objective:** Implement AES encryption and decryption.
- **Details:**
  - Support key sizes of 64, 128, or 256 bits.
  - Use a 64-bit block size for encryption.

### RSA Encryption Scheme
- **Objective:** Develop an RSA encryption and decryption scheme.
- **Details:**
  - Create public and private key pairs.
  - Encrypt and decrypt messages using RSA.

### MD5 Hash Algorithm
- **Objective:** Develop an MD5 hash algorithm.
- **Details:**
  - Calculate the Message Authentication Code (MAC) for messages.

### SHA Hash Algorithms
- **Objective:** Implement SHA-128 and SHA-256 hash algorithms.
- **Details:**
  - Compute the Message Authentication Code (MAC) for variable-sized messages.
  - Measure time consumption for varying message sizes for both SHA-128 and SHA-256.

### Digital Signature Standard (DSS)
- **Objective:** Develop DSS to verify legal communicating parties.
- **Details:**
  - Implement digital signature verification to ensure secure communication.

### Diffie-Hellman Key Exchange Protocol
- **Objective:** Design a Diffie-Hellman multiparty key exchange protocol.
- **Details:**
  - Perform a Man-in-the-Middle Attack simulation on the protocol.

### SSL Socket Communication
- **Objective:** Develop a client and server application using SSL socket communication.
- **Details:**
  - Implement secure communication between client and server using SSL.

### Telnet and Packet Analysis
- **Objective:** Develop a simple client-server model using telnet and analyze transmitted packets.
- **Details:**
  - Capture packets using `tshark` and analyze the data in pcap files.
  - Implement the scenario using SSH and observe encrypted data.

### JSON Web Token (JWT)
- **Objective:** Develop a web application implementing JSON Web Token (JWT).
- **Details:**
  - Use JWT for user authentication in the web application.
